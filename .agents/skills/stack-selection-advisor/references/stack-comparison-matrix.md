# Stack Comparison Matrix

## Frontend Frameworks

| Framework | Type | SEO | SSR | SSG | Learning | Ecosystem | Best For |
|-----------|------|-----|-----|-----|----------|-----------|----------|
| Next.js | React meta | Excellent | Yes | Yes | Medium | Massive | Full-stack, SEO |
| Nuxt | Vue meta | Excellent | Yes | Yes | Medium | Large | Vue teams, SEO |
| SvelteKit | Svelte meta | Good | Yes | Yes | Low | Growing | Performance, simplicity |
| Astro | Multi-framework | Excellent | Partial | Yes | Low | Growing | Content sites |
| Remix | React | Good | Yes | No | Medium | Moderate | Dynamic apps |
| Vite+React | SPA | Poor | No | No | Low | Massive | Dashboards, apps |

| Database | Type | Best For | Hosting |
|----------|------|----------|---------|
| PostgreSQL | Relational | Complex queries, relationships | Supabase, RDS, self-host |
| MySQL | Relational | Traditional apps, WordPress | PlanetScale, RDS |
| MongoDB | Document | Flexible schemas, rapid iteration | Atlas, self-host |
| SQLite | Embedded | Prototyping, small apps, mobile | File-based |
| Redis | Key-value | Caching, sessions, real-time | Redis Cloud, self-host |
