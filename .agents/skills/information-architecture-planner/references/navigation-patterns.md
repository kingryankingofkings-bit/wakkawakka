# Navigation Patterns

## Primary Navigation
- Horizontal top nav (most common)
- Hamburger menu on mobile (< 768px)
- Mega menu for sites with many sections
- Sticky nav for long pages

## Secondary Navigation
- Sidebar nav (documentation, dashboards)
- Tab navigation (related content sections)
- Breadcrumbs (deep hierarchies)
- In-page anchor nav (long content pages)

## Utility Navigation
- Top bar above main nav
- Includes: login, search, language, account

## Footer Navigation
- Sitemap links
- Legal links (privacy, terms)
- Social links
- Newsletter signup
