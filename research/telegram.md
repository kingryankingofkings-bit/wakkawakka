# Telegram Feature Research

This document compiles the feature research for Telegram, categorized into the reconciled 10-category taxonomy. Each category contains exactly 5 unique, specific, and actionable features, totaling 50 features.

---

## Category 1: Content Creation & Editing

### 1. In-App Media Editor

- **Category**: Content Creation & Editing
- **Description**: Provides advanced editing tools for photos and videos, including drawing, adding animated stickers, text, and adjusting brightness/contrast.
- **Sub-category / Actionable Details**: Media Editor. Attach a photo/video in a chat, tap the edit brush icon, adjust levels or apply stickers, and tap the checkmark.

### 2. Video Profile Avatars

- **Category**: Content Creation & Editing
- **Description**: Allows users to set a short looping video as their profile avatar instead of a static photo.
- **Sub-category / Actionable Details**: Dynamic Avatars. Go to Settings > Edit > Set Profile Photo or Video, capture or upload a video clip, select a frame for the static thumbnail, and save.

### 3. Custom Emoji Packs

- **Category**: Content Creation & Editing
- **Description**: Enables users to upload and use personalized custom emoji sets in chats, reactions, and statuses.
- **Sub-category / Actionable Details**: Personalized Expression. Open the Emoji Panel in any chat, tap the "+" icon or navigate to Settings > Stickers and Emoji, and import custom emoji sets.

### 4. Scheduled Messages

- **Category**: Content Creation & Editing
- **Description**: Sends messages automatically at a specified date and time in the future.
- **Sub-category / Actionable Details**: Delayed Message Delivery. Type a message in a chat, long-press the send button, select "Schedule message," choose the date and time, and tap Send.

### 5. Voice Transcription

- **Category**: Content Creation & Editing
- **Description**: Transcribes voice and video messages into readable text with a single tap.
- **Sub-category / Actionable Details**: Audio-to-Text. Tap the "->A" icon next to any incoming voice or video message to instantly generate a text transcription (Premium feature).

## Category 2: Content Discovery & Search

### 6. Public Broadcasting Channels

- **Category**: Content Discovery & Search
- **Description**: Allows creators to broadcast messages, media, and files to an unlimited number of subscribers.
- **Sub-category / Actionable Details**: Mass Broadcast. Tap the compose icon, select "New Channel," choose the name and set the visibility to "Public," creating a unique `@username` link.

### 7. Global Username Search

- **Category**: Content Discovery & Search
- **Description**: Finds public channels, groups, bots, and users by entering their unique `@` username in the search bar.
- **Sub-category / Actionable Details**: Directory Lookup. Type a username or keyword into the main search bar at the top of the chat list, and select from the "Global Search" results.

### 8. Customizable Chat Folders

- **Category**: Content Discovery & Search
- **Description**: Groups chat threads into distinct tabs based on custom filters like "Unread," "Channels," "Groups," or manually selected chats.
- **Sub-category / Actionable Details**: Chat Organization. Go to Settings > Chat Folders, select "Create New Folder," add chats or include/exclude categories, and name the folder.

### 9. People Nearby Discovery

- **Category**: Content Discovery & Search
- **Description**: Allows users to find and connect with other Telegram users and public groups physically close to them.
- **Sub-category / Actionable Details**: Location Discovery. Open the contact menu, select "Find People Nearby," grant location permissions, and toggle "Make Myself Visible" if desired.

### 10. Saved Messages Cloud

- **Category**: Content Discovery & Search
- **Description**: A personal cloud storage chat where users can forward important messages, store media, and keep notes.
- **Sub-category / Actionable Details**: Personal Cloud. Access via the side menu (or settings) under "Saved Messages," or forward any message from any chat to the "Saved Messages" contact.

## Category 3: Interpersonal & Community Engagement

### 11. Group Topic Threads

- **Category**: Interpersonal & Community Engagement
- **Description**: Organizes discussion within large groups by creating sub-topic channels to keep conversations structured.
- **Sub-category / Actionable Details**: Sub-Group Forums. In group settings (for groups with 100+ members), toggle "Topics" to on. Group admins and members can then create new threads.

### 12. Supergroups

- **Category**: Interpersonal & Community Engagement
- **Description**: Enables massive group chats hosting up to 200,000 members, featuring advanced moderation and pinning.
- **Sub-category / Actionable Details**: Mega Communities. Automatically upgrades standard groups as membership grows, allowing admins to pin messages, restrict members, and manage access.

### 13. Voice Chat Live Streams

- **Category**: Interpersonal & Community Engagement
- **Description**: Broadcasts live audio and video discussions to channels or groups with speaker controls and recording options.
- **Sub-category / Actionable Details**: Audio Spaces. Tap the group/channel profile header, tap the three-dot menu, select "Start Voice Chat" (or "Start Live Stream"), and invite participants.

### 14. Channel Discussion Groups

- **Category**: Interpersonal & Community Engagement
- **Description**: Links a public group chat to a broadcast channel, allowing subscribers to comment on and discuss channel posts.
- **Sub-category / Actionable Details**: Post Comments. Open channel settings, tap "Discussion," select an existing group or create a new one, and link them to enable commenting.

### 15. Interactive Polls & Quizzes

- **Category**: Interpersonal & Community Engagement
- **Description**: Creates multiple-choice polls, anonymous voting, or educational quizzes with single/multiple answers.
- **Sub-category / Actionable Details**: In-Chat Voting. Tap the attachment icon in a group/channel, select "Poll," write the options, toggle "Quiz Mode" if desired, and send.

## Category 4: Direct Messaging & Communication

### 16. Secret Chats

- **Category**: Direct Messaging & Communication
- **Description**: Provides end-to-end encrypted chats with self-destruct timers that leave no trace on Telegram's servers.
- **Sub-category / Actionable Details**: Secure Messaging. Open a contact's profile, tap the three-dot menu, select "Start Secret Chat," and set a self-destruct timer on messages.

### 17. Sent Message Editing

- **Category**: Direct Messaging & Communication
- **Description**: Allows users to edit the text of sent messages within 48 hours of dispatch.
- **Sub-category / Actionable Details**: Message Correction. Long-press a message you sent, tap the edit pencil icon, modify the text, and tap the checkmark to save edits.

### 18. Silent Message Dispatch

- **Category**: Direct Messaging & Communication
- **Description**: Sends a message to a recipient without playing a notification sound on their device.
- **Sub-category / Actionable Details**: Quiet Sending. Type a message, long-press the send button, and select "Send without sound" to deliver it silently.

### 19. Telescope Video Messages

- **Category**: Direct Messaging & Communication
- **Description**: Records and sends round video messages up to 60 seconds that stream instantly in-chat.
- **Sub-category / Actionable Details**: Circular Video notes. Tap the microphone icon once to switch to video, then press and hold the camera icon to record the video note.

### 20. In-Chat Translation

- **Category**: Direct Messaging & Communication
- **Description**: Translates messages written in foreign languages directly within the chat interface in real-time.
- **Sub-category / Actionable Details**: Inline Translator. Enable in Settings > Language > Show Translate Button. Tap a foreign message and select "Translate" from the context menu.

## Category 5: Monetization & E-Commerce

### 21. Telegram Stars Currency

- **Category**: Monetization & E-Commerce
- **Description**: A virtual currency used to pay for digital goods, services, and exclusive content offered by creators and bots.
- **Sub-category / Actionable Details**: In-App Payments. Purchase Stars via app store in-app purchases, then spend them inside bots, mini-apps, or paid channel subscription tiers.

### 22. Premium Paid Channels

- **Category**: Monetization & E-Commerce
- **Description**: Allows channel owners to lock exclusive content or posts behind a paid subscription or Star fee.
- **Sub-category / Actionable Details**: Paywalled Content. Enable monetization in channel settings, post media, set the price in Stars, and publish. Users must pay to unlock.

### 23. Sponsored Messages Ad Platform

- **Category**: Monetization & E-Commerce
- **Description**: Displays context-based, privacy-respecting sponsored posts at the bottom of public channels with large subscriber bases.
- **Sub-category / Actionable Details**: Privacy Ads. Created via `promote.telegram.org`. Advertisers target public channels based on topics without targeting individual user profiles.

### 24. In-App Payment Bots

- **Category**: Monetization & E-Commerce
- **Description**: Integrates bots that accept payments from credit cards and third-party gateways (Stripe, Google Pay) inside chats.
- **Sub-category / Actionable Details**: Conversational Commerce. Interact with a merchant bot, click checkout, input payment info via the integrated overlay, and complete the order.

### 25. Fragment TON Marketplace

- **Category**: Monetization & E-Commerce
- **Description**: A decentralized auction platform where users can buy and sell collectible usernames, anonymous phone numbers, and ads using TON cryptocurrency.
- **Sub-category / Actionable Details**: Decentralized Assets. Link a TON wallet to `fragment.com` to bid on usernames, transfer collectibles, or sell owned digital assets.

## Category 6: Analytics, Business & Creator Tools

### 26. Channel Growth Insights

- **Category**: Analytics, Business & Creator Tools
- **Description**: Displays detailed growth, views, languages, and interaction analytics for channels with more than 50 subscribers.
- **Sub-category / Actionable Details**: Stats Dashboard. Open the channel profile, tap the three-dot menu, select "Statistics" to view charts on subscriber flow and views.

### 27. Telegram Business Suite

- **Category**: Analytics, Business & Creator Tools
- **Description**: Unlocks professional features for business accounts, including greeting messages, away replies, custom quick replies, and location tags.
- **Sub-category / Actionable Details**: Business Dashboard. Subscribe to Telegram Premium or navigate to Settings > Telegram Business to customize operational settings.

### 28. Topic Engagement Metrics

- **Category**: Analytics, Business & Creator Tools
- **Description**: Tracks view count, message counts, and active member statistics inside individual group topics.
- **Sub-category / Actionable Details**: Topic Analytics. Group owners access topic statistics from the Group Info page > Statistics menu to audit discussion activity.

### 29. Interactive Bot API

- **Category**: Analytics, Business & Creator Tools
- **Description**: Allows developers to build highly interactive bots and mini-apps using Python, Node.js, and HTTP requests.
- **Sub-category / Actionable Details**: Developer Interface. Start a conversation with `@BotFather`, generate a bot token, and connect it to a server hosting the bot logic.

### 30. Bot Welcome Intros

- **Category**: Analytics, Business & Creator Tools
- **Description**: Configures custom start screens, media, and description texts that display before a user launches a bot.
- **Sub-category / Actionable Details**: Bot Onboarding. Customize bot profiles and start screens using command options inside `@BotFather` settings.

## Category 7: Privacy, Security & Safety

### 31. Auto-Delete Timers

- **Category**: Privacy, Security & Safety
- **Description**: Automatically deletes all messages in a selected chat after a duration of 24 hours, 7 days, or 1 month.
- **Sub-category / Actionable Details**: Auto-Destruct Chats. In chat settings, tap the three-dot menu, select "Auto-delete," choose the time interval, and apply to the chat.

### 32. Phone Number Hiding

- **Category**: Privacy, Security & Safety
- **Description**: Hides the user's phone number from all users or specific contacts, using the username as the primary identity.
- **Sub-category / Actionable Details**: Phone Masking. Go to Settings > Privacy and Security > Phone Number, and select "Nobody" to hide the number.

### 33. Password Session Lock

- **Category**: Privacy, Security & Safety
- **Description**: Secures access to the Telegram application using a local PIN or alphanumeric password.
- **Sub-category / Actionable Details**: App Lock Screen. Go to Settings > Privacy and Security > Passcode Lock, enable it, and set the PIN and auto-lock timeout.

### 34. Active Device Manager

- **Category**: Privacy, Security & Safety
- **Description**: Displays a list of all active sessions and devices logged into the account, with the ability to terminate sessions remotely.
- **Sub-category / Actionable Details**: Session Auditor. Go to Settings > Devices to review logged-in locations, IP addresses, and revoke any suspicious sessions.

### 35. Content Forward Protection

- **Category**: Privacy, Security & Safety
- **Description**: Restricts members from forwarding messages, copying text, or saving media from private groups or channels.
- **Sub-category / Actionable Details**: DRM Chats. Open group/channel settings, tap "Group Type" (or "Channel Type"), and toggle "Restrict saving content" to on.

## Category 8: Developer APIs & Integrations

### 36. Extensible Bot API

- **Category**: Developer APIs & Integrations
- **Description**: Provides rich widgets, custom keyboard overlays, and inline queries for third-party bots.
- **Sub-category / Actionable Details**: Bot Customization. Developers use JSON request payloads to send inline keyboard buttons, invoice forms, or custom media.

### 37. TDLib Open Source Library

- **Category**: Developer APIs & Integrations
- **Description**: An open-source library that implements all Telegram client features, allowing developers to build custom Telegram apps.
- **Sub-category / Actionable Details**: Client SDK. Integrate TDLib into C++, Java, C#, or Python projects to handle networking, local database storage, and cryptography.

### 38. MTProto Protocol API

- **Category**: Developer APIs & Integrations
- **Description**: A high-performance, secure protocol used for accessing the Telegram API from client applications.
- **Sub-category / Actionable Details**: Core Protocol. Connect custom client software to Telegram's production API gateway using MTProto schema handshakes.

### 39. In-Chat HTML5 Web Apps

- **Category**: Developer APIs & Integrations
- **Description**: Launches full-scale HTML5 web applications (Mini Apps) directly inside chat windows.
- **Sub-category / Actionable Details**: Web Interfaces. Register a Mini App via `@BotFather`, then launch it in-app using inline buttons or custom menu links.

### 40. Payment Gateway API

- **Category**: Developer APIs & Integrations
- **Description**: Connects Telegram bots to merchant payment gateways to handle credit card payments in-app.
- **Sub-category / Actionable Details**: Merchant Integration. Link provider accounts (e.g. Stripe) to the bot via `@BotFather` and construct payment payloads using the API.

## Category 9: Notifications & Time Management

### 41. Custom Mute Durations

- **Category**: Notifications & Time Management
- **Description**: Mutes chat notifications for a completely customizable duration, such as 2 hours, 3 days, or permanently.
- **Sub-category / Actionable Details**: Alert Muting. Long-press a chat, tap "Mute," select "Mute for..." and configure a custom duration using the date-time picker.

### 42. Notification Alert Exceptions

- **Category**: Notifications & Time Management
- **Description**: Overrides general notification settings to customize alerts for specific contacts or groups.
- **Sub-category / Actionable Details**: Exception Manager. Go to Settings > Notifications and Sounds, tap "Add Exception" under Private Chats or Groups, and select specific rules.

### 43. Smart Alert Limits

- **Category**: Notifications & Time Management
- **Description**: Sets limits on how often a chat can notify the user (e.g., maximum of 2 notifications every 10 minutes).
- **Sub-category / Actionable Details**: Frequency Control. Navigate to chat settings > Notifications > Customize, and set the alert frequency limit under the Smart Mute section.

### 44. Focus Filter Sync

- **Category**: Notifications & Time Management
- **Description**: Integrates with Apple Focus or Android Work modes to filter which Telegram notifications can wake the screen.
- **Sub-category / Actionable Details**: OS Focus Sync. Go to mobile OS Settings > Focus, add Telegram, and configure allowed contacts and notification priority tiers.

### 45. Non-Contact Auto-Archive

- **Category**: Notifications & Time Management
- **Description**: Automatically archives and mutes messages from new, unknown users who are not in the contact list.
- **Sub-category / Actionable Details**: Inbox Filter. Go to Settings > Privacy and Security > Archive and Mute, and toggle "Archive and Mute" to on.

## Category 10: Account Settings & Authentication

### 46. Multi-Account Switcher

- **Category**: Account Settings & Authentication
- **Description**: Allows users to log into up to three different phone numbers on the same Telegram application and switch between them instantly.
- **Sub-category / Actionable Details**: Multi-profile management. Open the side menu, tap the down arrow next to the username, select "Add Account," and input the phone number.

### 47. Username Domain Mapping

- **Category**: Account Settings & Authentication
- **Description**: Links custom domains or web addresses directly to a Telegram profile, channel, or group username.
- **Sub-category / Actionable Details**: Web Links. Format the link as `https://t.me/<username>`, which opens the target resource in the Telegram client or browser.

### 48. Cloud Two-Factor Password

- **Category**: Account Settings & Authentication
- **Description**: Requires an additional password when logging in from a new device, supplementing the standard SMS verification code.
- **Sub-category / Actionable Details**: Two-Step Verification. Go to Settings > Privacy and Security > Two-Step Verification, enter a password, set a hint, and provide a recovery email.

### 49. Auto-Self-Destruct Timer

- **Category**: Account Settings & Authentication
- **Description**: Automatically deletes the entire account, including all chats and media, if the user does not log in for a set period.
- **Sub-category / Actionable Details**: Account Lifespan. Go to Settings > Privacy and Security > Delete my account if away for, and select 1, 3, 6, or 12 months.

### 50. Large File Storage Tiers

- **Category**: Account Settings & Authentication
- **Description**: Allows sending and storing massive files up to 2GB (4GB for Premium users) in Saved Messages or chats.
- **Sub-category / Actionable Details**: File Repository. Tap the attachment icon, select "File," browse files, and upload. Files are stored in Telegram's secure cloud indefinitely.
