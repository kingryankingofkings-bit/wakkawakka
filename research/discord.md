# Discord Feature Research

This document compiles the feature research for Discord, categorized into the reconciled 10-category taxonomy. Each category contains exactly 5 unique, specific, and actionable features, totaling 50 features.

---

## Category 1: Content Creation & Editing

### 1. Discord Soundboard
- **Category**: Content Creation & Editing
- **Description**: Enables users to play short audio clips directly into voice channels, broadcasting the sound effects to all participants.
- **Sub-category / Actionable Details**: Voice Channel Audio Board. Join any voice channel, click the soundboard icon (resembling a speaker) on the bottom left overlay panel, and select a pre-loaded sound.

### 2. Discord Rich Text Markdown
- **Category**: Content Creation & Editing
- **Description**: Supports inline text styling and layouts inside channel chats using standard markdown notation like asterisks, tildes, and code blocks.
- **Sub-category / Actionable Details**: Chat Text Styling. Type asterisks around text for italics (e.g. `*italic*`), double asterisks for bold (`**bold**`), or triple backticks for formatted code blocks (` ```code``` `).

### 3. Discord Custom Emojis and Stickers
- **Category**: Content Creation & Editing
- **Description**: Allows server administrators to upload custom graphic images and animated stickers for members to use in messages.
- **Sub-category / Actionable Details**: Server Emoji Uploader. Go to Server Settings > Emoji. Click "Upload Emoji," select a local PNG or GIF file, and define its shortcode keyword.

### 4. Discord Clips
- **Category**: Content Creation & Editing
- **Description**: Records and saves recent gameplay and audio directly within a screen-share stream, saving it locally for quick post creation.
- **Sub-category / Actionable Details**: Stream Video Captures. Enable clipping in Settings > Clips. While streaming a game in a voice channel, press the designated hotkey (default: Alt+C) to save the last 30 seconds of video.

### 5. Discord Avatar Decorations
- **Category**: Content Creation & Editing
- **Description**: Allows users to customize their visual presence by applying animated, themed borders and effects to their profile picture.
- **Sub-category / Actionable Details**: Profile Visual Customization. Access User Settings > Profiles, click "Change Decoration," and choose from your purchased or unlocked animated borders.

## Category 2: Content Discovery & Search

### 6. Discord Server Discovery
- **Category**: Content Discovery & Search
- **Description**: A public directory of verified, partnered, and community-run Discord servers that users can browse by interest tags.
- **Sub-category / Actionable Details**: Community Directory. Click the compass icon at the bottom of the left-hand server sidebar, select a category, and type keywords in the search field to find public servers.

### 7. Discord Channel Search Filters
- **Category**: Content Discovery & Search
- **Description**: Offers a robust query search functionality inside text channels using criteria parameters like sender, dates, and media attachments.
- **Sub-category / Actionable Details**: Advanced Query Search. Tap the search bar in the top right, and select a filter chip like `from:username`, `has:file`, or `before:yyyy-mm-dd` to filter matching messages.

### 8. Discord Hubs for Students
- **Category**: Content Discovery & Search
- **Description**: Connects verified school email accounts to a localized network directory of student-run Discord servers.
- **Sub-category / Actionable Details**: Student Network Explorer. Click the compass icon, select "Student Hubs," enter your official school email address to receive a validation link, and browse matching student directories.

### 9. Discord Browse Channels Tab
- **Category**: Content Discovery & Search
- **Description**: A dedicated management view inside servers allowing members to subscribe to or hide channels from their left-hand sidebar list.
- **Sub-category / Actionable Details**: Server Sidebar Layout. Click the "Browse Channels" tab at the top of the server list, check or uncheck individual channels, or complete the server's onboarding questionnaire again.

### 10. Discord Server Search Bar
- **Category**: Content Discovery & Search
- **Description**: A localized search engine that parses messages, links, and user profiles strictly within the currently active server.
- **Sub-category / Actionable Details**: Server Content Index. Select a server, click the search box in the top right corner, type your query, and filter results by "Relevance" or "Recency."

## Category 3: Interpersonal & Community Engagement

### 11. Discord Role Hierarchy and Permissions
- **Category**: Interpersonal & Community Engagement
- **Description**: Establishes administrative rights and visual labels by creating and assigning server roles with custom priorities.
- **Sub-category / Actionable Details**: Server Role Customizer. Navigate to Server Settings > Roles. Click "Create Role," set a custom color, and toggle specific permission flags like "Manage Channels" or "Administrator."

### 12. Discord Stage Channels
- **Category**: Interpersonal & Community Engagement
- **Description**: Moderated audio rooms designed to host structured panel discussions, presentations, and QA sessions for large audiences.
- **Sub-category / Actionable Details**: Interactive Audio Events. Click the "+" button in your server's channel list, select "Stage Channel" as the type, set a title, and define the moderator roles before opening the room.

### 13. Discord Server Events
- **Category**: Interpersonal & Community Engagement
- **Description**: Allows administrators to schedule voice, video, or physical events with calendar reminder links for server members.
- **Sub-category / Actionable Details**: Digital Event Organizer. Click the server name at the top of the sidebar, select "Create Event," fill in the venue type (e.g. voice channel or external location), time, and description, and click "Create."

### 14. Discord Forum Channels
- **Category**: Interpersonal & Community Engagement
- **Description**: Provides structured, message-board-style discussion spaces where users create separate posts with tag labels and comments.
- **Sub-category / Actionable Details**: Forum Thread Boards. Click the "+" button in the channel list, select "Forum Channel," define tag labels (e.g. "Support" or "Chat"), and set guidelines for new posts.

### 15. Discord Member Onboarding Screens
- **Category**: Interpersonal & Community Engagement
- **Description**: Displays welcome popups, rules verification steps, and channel customization questionnaires for new members.
- **Sub-category / Actionable Details**: Server Join Questionnaire. Access Server Settings > Safety Setup > Onboarding. Define default channels, create member profile questions, and toggle "Rules Screening" to require verification.

## Category 4: Direct Messaging & Communication

### 16. Discord Direct Message Voice Calls
- **Category**: Direct Messaging & Communication
- **Description**: Allows users to initiate private, high-quality audio or video calls with a friend outside of any server environment.
- **Sub-category / Actionable Details**: Direct Audio Contacts. Open your Direct Messages list, select a conversation thread, and click the phone or camera icon in the top right corner to start a call.

### 17. Discord DM Group Chats
- **Category**: Direct Messaging & Communication
- **Description**: Creates private, invitation-only text and voice channels for up to 10 friends, bypasses server setup completely.
- **Sub-category / Actionable Details**: Private Group Channels. Click the "+" icon at the top of your Direct Messages list, select the contacts you want to add, and click "Create Group DM."

### 18. Discord Message Threading
- **Category**: Direct Messaging & Communication
- **Description**: Creates nested, sub-channel conversations stemming from a single message to keep chat topics organized.
- **Sub-category / Actionable Details**: Nested Chat Threads. Hover over any message in a text channel, click the "#" thread icon, name the thread, and send a message to open the sidebar thread window.

### 19. Discord Pin Message
- **Category**: Direct Messaging & Communication
- **Description**: Saves crucial posts or links in a dedicated channel registry, making them easily retrievable by members.
- **Sub-category / Actionable Details**: Anchored Messages. Hover over a message, click the three-dot menu, and select "Pin Message." View all pinned messages by clicking the pushpin icon in the channel header.

### 20. Discord Voice Message Recording
- **Category**: Direct Messaging & Communication
- **Description**: Records and sends short audio voice messages directly into a direct message or server chat.
- **Sub-category / Actionable Details**: Audio Messaging. Press and hold the microphone icon in the chat input bar on mobile, record your voice note, and release the icon to instantly send it.

## Category 5: Monetization & E-Commerce

### 21. Discord Server Subscriptions
- **Category**: Monetization & E-Commerce
- **Description**: Allows server owners to create paid tiers that grant members custom roles, special channels, and custom badges.
- **Sub-category / Actionable Details**: Creator Monetization Hub. Go to Server Settings > Server Subscriptions. Connect a Stripe account, set up pricing tiers, and define the specific role permissions for each tier.

### 22. Discord Nitro Subscription
- **Category**: Monetization & E-Commerce
- **Description**: A monthly paid upgrade service that boosts individual user capabilities across the entire Discord platform.
- **Sub-category / Actionable Details**: Premium Tier Subscription. Access User Settings > Nitro. Select a plan (Nitro or Nitro Basic) and complete payment to unlock features like 500MB uploads, HD streaming, and custom emoji usage.

### 23. Discord App Shop
- **Category**: Monetization & E-Commerce
- **Description**: An in-app storefront where users can buy premium avatar decorations, profile effects, and soundboard sounds.
- **Sub-category / Actionable Details**: Profile Cosmetics Store. Click the "Shop" tab at the top of your Direct Messages list, browse the themes (e.g. Anime, Fantasy), and purchase decorations using a linked credit card.

### 24. Discord Server Boosting
- **Category**: Monetization & E-Commerce
- **Description**: Allows members to purchase boosts that collectively upgrade server performance, audio quality, and custom slot allowances.
- **Sub-category / Actionable Details**: Server Upgrades. Click on a server's name, select "Server Boost," select the number of boosts to apply, and complete the check-out process.

### 25. Discord One-Time Purchases
- **Category**: Monetization & E-Commerce
- **Description**: Integrates checkout mechanisms for buying digital tickets, games, or exclusive content inside developer applications.
- **Sub-category / Actionable Details**: App Monetization Store. Interact with a developer bot or app interface in a server, select an item to purchase (e.g. a game expansion), and complete payment via Discord's payment modal.

## Category 6: Analytics, Business & Creator Tools

### 26. Discord Server Insights
- **Category**: Analytics, Business & Creator Tools
- **Description**: Provides server owners with a data dashboard tracking member growth, active communicators, and long-term retention.
- **Sub-category / Actionable Details**: Community Analytics Dashboard. Accessible under Server Settings > Server Insights on servers with over 500 members. Generates CSV exports tracking activity over time.

### 27. Discord Announcement Channels
- **Category**: Analytics, Business & Creator Tools
- **Description**: Specialized text channels that allow admins to broadcast posts to external servers that choose to follow the channel.
- **Sub-category / Actionable Details**: Broadcast Syndication. Click the "+" button in the channel list, select "Announcement Channel," write a post, and click "Publish" to push it to all subscribing servers.

### 28. Discord Creator Portal
- **Category**: Analytics, Business & Creator Tools
- **Description**: A web resource containing tutorials, analytics guides, and monetization optimization tools for server managers.
- **Sub-category / Actionable Details**: Creator Learning Center. Access via `discord.com/creator-portal` using your Discord account credentials to read optimization case studies and join the official Creator Server.

### 29. Discord Welcome Screen Customizer
- **Category**: Analytics, Business & Creator Tools
- **Description**: Builds a clean, visual landing popup page displaying recommended channels and descriptions for new visitors.
- **Sub-category / Actionable Details**: Welcome Banners. Open Server Settings > Welcome Screen. Click "Set up Welcome Screen," choose target channels, add descriptions, and upload a custom server description.

### 30. Discord Server Webhook Analytics
- **Category**: Analytics, Business & Creator Tools
- **Description**: Tracks delivery rates, latency, and response codes for webhook integration payloads inside server channels.
- **Sub-category / Actionable Details**: API Webhook Monitoring. Under Server Settings > Integrations > Webhooks. Check the status log of individual webhooks to verify programmatic payload traffic.

## Category 7: Privacy, Security & Safety

### 31. Discord AutoMod
- **Category**: Privacy, Security & Safety
- **Description**: An automated content moderation engine that scans, blocks, or flags messages containing blocked keywords or spam.
- **Sub-category / Actionable Details**: Server Filter Rules. Open Server Settings > Safety Setup > AutoMod. Click "Create Rule," define the trigger keywords, choose action alerts, and set up exemption roles.

### 32. Discord Block & Mute List
- **Category**: Privacy, Security & Safety
- **Description**: Prevents designated accounts from direct messaging you, sending friend requests, or showing up in text and voice channels.
- **Sub-category / Actionable Details**: User Relationship Management. Right-click on a user's name or avatar profile card, and select "Block" or "Mute" from the context menu list.

### 33. Discord Friend Request Filters
- **Category**: Privacy, Security & Safety
- **Description**: Restricts the range of users who are allowed to send you incoming friend invitations.
- **Sub-category / Actionable Details**: Request Permission Toggles. Access User Settings > Friend Requests. Toggle off "Everyone," "Friends of Friends," or "Server Members" to adjust privacy levels.

### 34. Discord Safe Direct Messaging
- **Category**: Privacy, Security & Safety
- **Description**: Scans and automatically deletes direct messages that contain explicit media, files, or links.
- **Sub-category / Actionable Details**: Media Security Scanner. Access User Settings > Privacy & Safety. Select between "Filter direct messages from everyone," "Filter direct messages from non-friends," or "Do not filter."

### 35. Discord Verification Levels
- **Category**: Privacy, Security & Safety
- **Description**: Server-wide safety requirements that set criteria for how long an account must be registered or verified before posting.
- **Sub-category / Actionable Details**: Server Entry Constraints. Navigate to Server Settings > Safety Setup > Verification Level. Choose levels ranging from "None" to "Very High" (requires verified phone number).

## Category 8: Notifications & Time Management

### 36. Discord Server Mute Options
- **Category**: Notifications & Time Management
- **Description**: Mutes message alerts and push notifications for an entire server, category, or channel.
- **Sub-category / Actionable Details**: Alert Silencing. Right-click on a server icon or channel name, select "Mute Server" or "Mute Channel," and choose a duration from 15 minutes to "Until I turn it back on."

### 37. Discord Desktop Notification Settings
- **Category**: Notifications & Time Management
- **Description**: Controls unread indicators, push notification sounds, text-to-speech alerts, and system-level alerts.
- **Sub-category / Actionable Details**: Sound & Push Alerts. Navigate to User Settings > Notifications. Toggle the settings for "Enable Desktop Notifications," "Text-to-Speech," and individual event sounds.

### 38. Discord Notification Overrides
- **Category**: Notifications & Time Management
- **Description**: Establishes custom alerts for specific search keywords, phrases, or individual user tags within a server.
- **Sub-category / Actionable Details**: Keyword Alerts. Click the server name, select "Notification Settings," scroll down to "Notification Overrides," and add custom channels or categories with distinct alert rules.

### 39. Discord Push Notification Timeout
- **Category**: Notifications & Time Management
- **Description**: Sets the idle time threshold that must pass before desktop notifications are redirected to mobile push alerts.
- **Sub-category / Actionable Details**: Idle Notification Sync. Navigate to User Settings > Notifications, scroll to "Push Notification Active Timeout," and set a duration (e.g. 10 minutes).

### 40. Discord Do Not Disturb Mode
- **Category**: Notifications & Time Management
- **Description**: Changes your user presence status to suppress all incoming sound alerts and desktop popups globally.
- **Sub-category / Actionable Details**: Status Muting. Click your avatar icon in the bottom left corner, select the status submenu, and click "Do Not Disturb" to silence alerts.

## Category 9: Developer APIs & Integrations

### 41. Discord Gateway API
- **Category**: Developer APIs & Integrations
- **Description**: A WebSockets-based API that streams real-time events like message creations, voice status changes, and server updates to developer clients.
- **Sub-category / Actionable Details**: Real-time Event Stream. Developers configure a WebSocket client connection to the URL `wss://gateway.discord.gg` using their bot account token to receive JSON event payloads.

### 42. Discord REST API
- **Category**: Developer APIs & Integrations
- **Description**: An HTTPS API that allows developers to programmatically manage servers, channels, messages, roles, and user profiles.
- **Sub-category / Actionable Details**: Programmatic Server Actions. Make authorized HTTP requests to base endpoint `https://discord.com/api/v10` (e.g. `POST /channels/{channel.id}/messages` to send a message).

### 43. Discord Interactions API
- **Category**: Developer APIs & Integrations
- **Description**: Receives user-triggered actions (like slash commands, buttons, and select menus) and sends responses via HTTP webhooks.
- **Sub-category / Actionable Details**: Interactive App Backend. Developers configure an interactions endpoint URL on the Discord Developer Portal. The server receives POST payloads and must respond within 3 seconds.

### 44. Discord Embedded App SDK
- **Category**: Developer APIs & Integrations
- **Description**: An SDK that enables developers to build web apps and multiplayer games that run directly inside Discord voice channel activities.
- **Sub-category / Actionable Details**: Voice Activity Development. Developers import the `@discord/embedded-app-sdk` package in their web app, calling initialization routines to authenticate and sync states with voice sessions.

### 45. Discord OAuth 2.0 Identity Endpoint
- **Category**: Developer APIs & Integrations
- **Description**: Authenticates users and requests access permissions for their server lists, connections, and profile details.
- **Sub-category / Actionable Details**: Federated Identity Auth. Developers configure authorization redirect links pointing to `/api/oauth2/authorize` with requested scopes (e.g. `identify guilds`).

## Category 10: Account Settings & Authentication

### 46. Discord Two-Factor Authentication
- **Category**: Account Settings & Authentication
- **Description**: Secures account access by requiring a TOTP security code from an authenticator app when logging in.
- **Sub-category / Actionable Details**: Account Protection. Navigate to User Settings > My Account, click "Enable Two-Factor Auth," scan the QR code with an authenticator app, and save the backup codes.

### 47. Discord Authorized Apps
- **Category**: Account Settings & Authentication
- **Description**: Displays a history of third-party applications that have been granted access to your account details, with options to revoke access.
- **Sub-category / Actionable Details**: Third-Party Integration Audit. Navigate to User Settings > Authorized Apps. Click "Deauthorize" next to any app to revoke its access tokens.

### 48. Discord User Connections
- **Category**: Account Settings & Authentication
- **Description**: Integrates external accounts like Spotify, Steam, Xbox, and PlayStation to display game status and activity on profiles.
- **Sub-category / Actionable Details**: Social Presence Linking. Go to User Settings > Connections. Click on a service icon, log in to verify ownership, and toggle "Display on profile."

### 49. Discord Account Deletion and Disable
- **Category**: Account Settings & Authentication
- **Description**: Allows users to temporarily deactivate their accounts or permanently delete their user data and servers.
- **Sub-category / Actionable Details**: Account Offboarding. Open User Settings > My Account. Click "Delete Account" or "Disable Account," enter your password and 2FA code to confirm the action.

### 50. Discord Security Keys
- **Category**: Account Settings & Authentication
- **Description**: Allows users to register hardware keys or biometric logins for passwordless authentication.
- **Sub-category / Actionable Details**: Hardware Credential Manager. Go to User Settings > My Account. Click "Register a Security Key" under the password security section and follow the browser prompts.
